# 手动下载封面功能

## 功能概述

为Harmony音乐播放器添加了手动下载专辑封面的功能。用户可以选择歌曲，从多个在线来源（iTunes、MusicBrainz、Last.fm）下载封面，并预览后保存到数据库。

## 主要特性

### 1. 封面下载对话框
- **多选支持**: 可以为单首或多首歌曲下载封面
- **封面预览**: 下载前预览封面效果
- **多源支持**: 支持 iTunes、MusicBrainz、Last.fm 三个封面来源
- **批量处理**: 可在不同歌曲间切换，依次下载封面
- **实时反馈**: 显示下载进度和状态信息

### 2. 用户界面
- **现代化设计**: 采用与播放器一致的暗色主题
- **直观操作**: 通过下拉菜单选择歌曲和封面来源
- **状态提示**: 清晰的成功/失败消息提示
- **响应式布局**: 适配不同屏幕尺寸

### 3. 技术实现
- **线程安全**: 使用 QThread 进行异步下载，不阻塞UI
- **错误处理**: 完善的异常处理和用户友好的错误提示
- **缓存管理**: 自动管理封面缓存，避免重复下载
- **数据库集成**: 下载的封面自动保存到数据库

## 使用方法

### 通过右键菜单
1. 在音乐库中选中一首或多首本地歌曲
2. 右键点击打开上下文菜单
3. 选择"下载封面"选项
4. 在对话框中：
   - 使用下拉菜单选择要下载封面的歌曲
   - 选择封面来源（iTunes、MusicBrainz、Last.fm）
   - 点击"下载"按钮开始下载
   - 预览封面效果
   - 满意后点击"保存"按钮

### 对话框功能
- **歌曲选择**: 顶部下拉菜单显示所有选中的歌曲
- **封面来源**: 中间下拉菜单可选择封面来源
- **下载按钮**: 点击开始从选定来源下载封面
- **封面预览**: 中央显示区展示下载的封面图片
- **保存按钮**: 下载成功后，点击保存到数据库

### 关键类

#### CoverDownloadDialog
主对话框类，负责：
- UI 布局和事件处理
- 歌曲信息显示
- 封面预览
- 保存到数据库

#### CoverDownloadThread
工作线程类，负责：
- 异步下载封面
- 避免阻塞UI
- 下载完成通知

## 数据流

1. **用户选择歌曲** → LibraryView 右键菜单
2. **打开对话框** → CoverDownloadDialog 初始化
3. **选择封面来源** → 用户选择 iTunes/MusicBrainz/Last.fm
4. **下载封面** → CoverDownloadThread 异步下载
5. **预览封面** → 在对话框中显示图片
6. **保存封面** → 保存到缓存并更新数据库
7. **事件通知** → EventBus 通知其他组件更新
